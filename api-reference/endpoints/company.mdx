---
title: "LinkedIn Company Scraper"
api: "GET https://veille.io/api/v1/scraper/company"
description: "This endpoint allows you to scrape a LinkedIn company."
---

### Header

<ParamField header="x-api-key" type="string" required={true}>
    This parameter specifies the private key you'll need for Veille.io access.
</ParamField>

### Parameters

#### Search Query

<ParamField query="query" type="string" required={true} default="https://www.linkedin.com/company/airbusgroup">
    Linkedin company URL : `https://www.linkedin.com/company/companygroup`
</ParamField>
